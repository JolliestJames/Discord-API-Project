How To Make a Discord Bot with Discord.js

For many gamers, Discord has become a mainstay for communication with others who enjoy video games. The platform not only makes for easy voice and text chat, but allows users to build servers that can host large gaming communities. One of the best parts of Discord is the ability to make a bot that help keep a server organized, running smoothly and even be a source of entertainment for server members. Such a bot can be made with the help of Discord.js, a node.js library that uses the Discord API to make the creation of a bot with JavaScript possible.

What You'll Need:

  • Discord's API Documentation
  • Discord.js Library Documentation
  • A basic knowledge of JavaScript
  • A basic grasp on how to use the command line
  • The lastest version of node.js downloaded and installed
  • The lastest version of Discord downloaded and installed, along with your own Discord account

And that's it! There are quite a few steps to get through, so let's get started.
