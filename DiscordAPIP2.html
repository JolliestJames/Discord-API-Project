Get Discord.js

First thing's first. Before anything fun happens, we need to use node.js to create a package.json file and get the Discord.js library.

Assuming you have node.js installed, use your command line to navigate to the directory where you'd like to store your bot's source code. Enter the command below into the command line:

npm init

Now we should install the Discord.js library. Enter this next command into your command line as well:

npm install discord.js --save

Easy enough, right? If you take a look at your package.json file, you should see discord.js in the dependencies object. If not, something went wrong.
